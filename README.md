## About

Simple jQuery plugin to extend the selection of callable CSS colour names.

### Code examples

```bash
	$("body").customColour({
		colour: "burgundy",
		property: "backgroundColor"
	})
```

```bash
	$("body").customColour({
		colour: "burgundy",
		property: "background"
	})
```

```bash
	$("body").customColour({
		colour: "burgundy",
		property: "color"
	})
```
